Penrose Pixels: Super-Resolution in the Detector Layout Domain

We present a novel approach to reconstruction based super-resolution that explicitly models the detector's pixel layout and a new variant of the well known error back-projection super-resolution algorithm that makes use of the exact detector model in its back projection operator for better accuracy.

Detecting Doctored JPEG Images via DCT Coefficient Analysis

We propose an approach that can detect doctored JPEG images and further locate the doctored parts, by examining the double quantization effect hidden among the DCT coefficients.

First Order Approximation for Texture Filtering

In this paper, we achieve high quality antialiasing by using signal processing and sampling theory and proposing a first order approximating filter.

Real-Time Rendering of Realistic Rain

To render realistic rain in real-time, videos of real rain are analyzed to extract the rain mattes; random samples of these rain stroke mattes are then used for online synthesis. We incorporate pre-computed radiance transfer (PRT) in our particle system to take into account the scene radiance.

Detecting Doctored Images Using Camera Response Normality and Consistency

To differentiate real images/videos from doctored ones, we propose an approach that computes the response functions of the camera by selecting appropriate patches in different ways.

On-Line Signature Verification With Two-Stage Statistical Models

To make signature verification more accurate, we construct a system, which is composed of a simplified GMM model for global signature features, and a discrete HMM model for local signature features.

Off-line Signature Verification Incorporating the Prior Model

This paper presents an off-line signature verification system that only requires the genuine signatures of a new user.

Signature verification using integrated classifiers

This paper presents a new approach for off-line signature verification with Boosting algorithm.

Correction and Rectification of Light Fields

We propose a correction and rectification framework that uses only four images to capture an ideal light field.

On the Fundamental Limits of Reconstruction-Based Super-resolution Algorithms

This paper provides explicit limits for a major class of super-resolution algorithms, called reconstruction-based algorithms, under both real and synthetic conditions.
